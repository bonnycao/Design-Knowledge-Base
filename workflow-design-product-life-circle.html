A Screen Flow is UI-level: it shows how a user moves from screen to screen. It often looks like a sitemap with arrows, or a set of screens connected by navigation paths.

  • Useful for planning navigation and information architecture
  • Very common in mobile apps and SaaS dashboards
  • Can evolve into a Wireflow if you connect actual wireframes

You don‚Äôt need all of these for every project. Choose based on risk, complexity, and who needs alignment.

Quick picks
